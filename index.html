LiveCode

LiveCode is a cross-platform rapid-application environment inspired by HyperCard, using an English-like xTalk scripting language. It offers a compile-free workflow so the same code runs across desktop, mobile and the web, speeding prototyping and release. Its readable, expressive syntax emphasizes maintainability and productivity.
