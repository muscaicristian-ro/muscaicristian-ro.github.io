This is my retro-styled developer portfolio. Inspired by the early days of programming with GW-BASIC, Pascal, and FoxPro.

LiveCode

LiveCode is a cross-platform rapid-application environment inspired by HyperCard, using an English-like xTalk scripting language. It offers a compile-free workflow so the same code runs across desktop, mobile and the web, speeding prototyping and release. Its readable, expressive syntax emphasizes maintainability and productivity.

VB.NET

VB.NET is a modern, object-oriented dialect of Visual Basic that runs on the .NET platform. It uses a readable, English-like syntax and integrates tightly with the .NET libraries, making desktop, web, and service development straightforward. With strong typing, async/await, LINQ, and first-class tooling in Visual Studio, VB.NET balances approachability with production-grade power.

Visual Basic Classic

Visual Basic 1.0–6.0 (“classic VB”) made Windows application building delightfully fast. With a drag-and-drop form designer, event-driven code, and rich controls, it turned ideas into running EXEs in hours, not weeks. Its readable Basic syntax, COM/ActiveX component model, and database wizards powered countless business apps. VB6 crowned the era, balancing simplicity with serious capability that still inspires tools today.

Legacy / Early Projects

These legacy projects celebrate the roots of home computing: GW-BASIC line-numbered programs that PEEK/POKE memory and make the PC speaker sing; Turbo Pascal’s lightning-fast compiler and tidy Borland IDE; FoxPro’s xBase databases with blistering indexes and quick forms; and straight Assembly, where interrupts and addresses were your API. Built on 8086/286/386 machines, CRTs, and floppy disks, they honor constraints—kilobytes not gigabytes, 16-bit segments, and code that had to start instantly and never crash. Here you’ll find small utilities, experiments, and tiny games that squeezed surprising behavior from modest hardware. It’s a tribute to the tools and tinkerers that taught clarity, efficiency, and craftsmanship—and a reminder that today’s stacks stand on the shoulders of those humming, green-glow machines.
